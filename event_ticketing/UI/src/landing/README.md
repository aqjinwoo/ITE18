# Tixaro Landing Page

This folder contains the landing page for the Tixaro event ticketing system.

## Files

- **LandingPage.tsx** - Main landing page component with all sections

## Design Features

The landing page follows the same design system as the main application:
- **Dark Theme** - Black background with dark cards
- **Electric Blue Accents** - Primary color for CTAs and highlights
- **Clean Typography** - Bold headings with proper hierarchy
- **Smooth Animations** - Hover effects and transitions
- **Fully Responsive** - Mobile-first design

## Sections

### 1. Navigation Bar
- Sticky navigation with blur effect
- Logo with Tixaro branding
- Navigation links to page sections
- Sign In / Get Started buttons

### 2. Hero Section
- Large headline with gradient text
- Call-to-action buttons
- Statistics showcase (50K+ customers, 1000+ events, etc.)
- Visual grid with event images

### 3. Featured Events
- Grid of 3 featured events
- Event cards with images, dates, locations, prices
- "View All Events" CTA

### 4. Browse by Category
- 5 category cards (Music, Sports, Arts, Technology, Entertainment)
- Icon-based visual design
- Event count per category

### 5. How It Works
- 3-step process explanation
- Visual icons for each step
- Simple, clear descriptions

### 6. Why Choose Tixaro
- 4 key features showcase
- Icons and descriptions
- Payment methods display (Credit Card, Debit Card, GCash, PayPal, PayMaya)

### 7. Call-to-Action Section
- Final conversion section
- Large headline with gradient
- Dual CTA buttons

### 8. Footer
- Brand information
- Quick links
- Support links
- Contact information
- Copyright and legal links

## Usage

The landing page is automatically shown when the app loads. Users can navigate to:
- **Sign In** - Login page
- **Get Started** - Registration page
- **Browse Events** - Requires registration/login
